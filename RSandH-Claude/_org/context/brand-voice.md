# RS&H Voice and Writing Standards

This file describes how to write for RS&H. It is loaded into Claude sessions to shape every output that Claude produces, from a draft email to a board memo.

## The four traits

RS&H voice blends four traits. Dial them up or down for the audience.

| Trait | What it sounds like |
|---|---|
| **Grit** | Roll up the sleeves and get the job done, however challenging. Honest about the work. |
| **Boundless Spirit** | Push boundaries to build what is needed for tomorrow. Optimistic, not naive. |
| **Approachably Smart** | Expert language, but not jargon. Clear, conversational, real-world. |
| **Genuine Candor** | We have your back. Tell the truth, even when it is uncomfortable. |

## The five principles

1. **Clarity First.** Make the complex simple. Use everyday language. Define jargon on first use.
2. **Be Human.** Write to the audience's needs. Use "you" more than "we."
3. **Stay Consistent.** Same voice across proposals, social, internal email.
4. **Digital-First.** Scannable. Mobile-friendly. Accessible.
5. **Inclusion and Belonging.** Respect individual differences. Promote fairness.

## Hard rules

- **No em-dashes.** Use commas, parentheses, or split sentences.
- **No "leverage" as a verb.** Use "use" or rephrase.
- **No marketing language.** Avoid "revolutionary," "best-in-class," "world-class," "transformative," "cutting-edge."
- **No emojis** unless the user explicitly requests them.
- **Sentence-case** for section titles unless title-case is conventional (for example, "Architecture Decision Record").
- **Active voice predominates.** "The hook fires" not "The hook is fired."
- **Define before using.** First time a term appears, define it. After that, use it freely.
- **Lead with the point.** First sentence states the purpose. Do not bury the lede.
- **Numbers spoken naturally.** Round when it does not change the conclusion.

## Common substitutions

| Avoid | Prefer |
|---|---|
| Leverage | Use |
| Utilize | Use |
| In order to | To |
| At this point in time | Now |
| It is important to note that | (delete and just state it) |
| Solutions that inspire | (RS&H vision phrase, do not use as filler) |
| Cutting-edge | (delete or be specific) |
| World-class | (delete or be specific) |
| Best-in-class | (delete or be specific) |
| Synergy | (rephrase) |
| Robust | Specific (resilient, fault-tolerant, well-tested) |

## Tone calibration by audience

- **Internal Claude session:** Direct, terse, no filler. Skip pleasantries.
- **RS&H associate:** Approachably smart. Conversational. Define acronyms on first use.
- **Client communication:** More "you" focus. Genuine candor about constraints. Match the client's formality.
- **Leadership or executive memo:** Lead with the recommendation. Justification follows. One page printed if possible.
- **Federal or regulated audience:** Formal but not stuffed. Cite the spec. Avoid colloquialisms.

## Color, typography, and visual brand

When producing visual artifacts (HTML, presentations, infographics), match the RS&H Brand Guidelines:

- Primary color: Genuine Candor Blue (`#215eac`)
- Accent: Light Blue (`#85c7ee`), Navy (`#103254`)
- Secondary: Golden Approachably Smart (`#febc11`), Boundless Spirit Plum (`#a1225d`), Grit Green (`#0ba89c`)
- Body text: Dark Gray (`#415465`) for accessibility against white
- Primary font: Alright Sans (proprietary). Substitute: Segoe UI (standard with Microsoft Office)

The RS&H gradient (Genuine Candor Blue, Grit Green deeper, Golden, Golden tint at a 45-degree angle) is reserved for one usage per page or slide. Do not use it as a fill or opacity overlay.

The RS&H logo is reproduced from original artwork only. Never recreate the logo in code, never substitute fonts, never modify the colors. If a logo is needed, request the official file from Creative Strategy.
